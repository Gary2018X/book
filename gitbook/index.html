gitbook

Gitbook 使用入门

GitBook 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。

本书将简单介绍如何安装、编写、生成、发布一本在线图书。 gitbook 生成电子书主要有三种方式:

gitbook-cli 命令行操作，简洁高效，适合从事软件开发的相关人员. gitbook-editor 编辑器操作，可视化编辑，适合无编程经验的文学创作者.但是gitbook editor的注册和登录需要科学上网. gitbook.com 官网操作，在线编辑实时发布，适合无本地环境且科学上网的体验者.
